Inventory Management System
This is a simple Python program to manage shoe inventory. It allows users to:
Add new shoes to the inventory
View all shoes in a table format
Re-stock items with low quantity
Search for shoes by code
Calculate total value of items
Find the highest quantity item for sale


Files
inventory.py → Main Python program for inventory management
inventory.txt → Stores shoe data in txt format

How to Use
Make sure you have Phyton installed.
Install tabulate library if not already installed
Run the program: inventory.py

Follow the menu to perform actions:
A : Add new shoe
B : View all shoes
C : Re-stock shoes
D : Search shoe by code
E : View value of items
F : Highest quantity item
Q : Quit
